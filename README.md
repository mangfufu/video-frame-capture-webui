# 本地媒体工具

一个零依赖的本地前端项目，直接在浏览器中打开即可使用。

## 功能

- 支持拖拽或选择本地视频文件
- 支持批量导入视频并切换当前视频
- 支持导出当前帧
- 支持按秒级间隔批量截帧
- 支持导出为 `PNG`、`JPG`
- 支持图片局部模糊页面
- 支持手动画蒙版、橡皮擦、羽化、模糊强度、不透明度设置

## 使用方式

1. 进入 `video-frame-capture-app` 文件夹。
2. 直接用浏览器打开 `index.html`。
3. 视频截帧使用 `index.html`，图片局部模糊使用 `image-blur.html`，两个页面也可互相跳转。
4. 调整时间轴后点击“导出当前帧”，或设置间隔后点击“按间隔批量导出”。
5. 图片模糊页中上传图片后，直接在画布上绘制蒙版，再导出图片。

## 说明

- 当前实现为浏览器直接下载到本地，不依赖后端。
- 批量导出大量图片时，浏览器可能触发多文件下载提示，这是浏览器安全策略，不是程序限制。
